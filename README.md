### IAM Org Creator

## Overview

Service allowing users to create an organisation on demand

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Tools you will need to setup this service

```
npm version 6+
nodejs version 10+
```

## Config

[configuration variables](scr/doc/config.md)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
